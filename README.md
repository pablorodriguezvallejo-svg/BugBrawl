# Bug Brawl

Bug Brawl es una página web desarrollada como parte de mi TFG. El proyecto presenta un juego de cartas de insectos, combinando combates, estrategia y una estética inspirada en la naturaleza.

La web ha sido diseñada para mostrar el universo del juego mediante diferentes secciones, personajes y elementos interactivos, buscando crear una interfaz atractiva, responsive y fácil de navegar.

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript

## Funcionalidades

* Diseño responsive para ordenador, tablet y móvil.
* Menú de navegación adaptado.
* Galería de imágenes con vista ampliada.
* Cartas interactivas con animación de giro.
* Animaciones al hacer scroll.
* Componentes visuales personalizados.

## Estructura del proyecto

* `index.html` → estructura principal de la página.
* `galeria.html` → página de la galeria.
* `styles.css` → archivo de estilos.
* `script.js` → scripts y funcionalidades.
* `imagenes/` → carpeta con los recursos gráficos.
* `fuentes/` → tipografías utilizadas.
